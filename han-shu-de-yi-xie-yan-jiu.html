在创建函数的时候，我们可以使用函数表达式，函数声明，new 函数，和箭头函数方式的创建函数，但是我们在创建对象内部函数的时候又额外可以使用{key(){}}的方式创建函数。

这篇文章主要是对对象内容的多种函数创建方式的研究。

# this指向

首先我们创建一个对象：

let obj = {
  a: function(){
    console.log(this)
  },
  b(){
    console.log(this)
  },
  c:()=>{
		console.log(this)  
	}
}

我们运行一下obj.a() obj.b() obj.c()可以得到如下打印机内容：

// obj.a()
{a: ƒ, b: ƒ, c: ƒ}
// obj.b() 
{a: ƒ, b: ƒ, c: ƒ}
// obj.c()
Window {window: Window, self: Window, document: document, name: '', location: Location, …}

我们可以看到函数a和函数b的this指向都是对象obj，但是箭头函数c的this指向了Window对象，这是根据以往的普通函数和箭头函数的区别，我们知道现在的this指向是没有问题的。

# 更改this指向

我们使用call更改this指向为{},执行得到如下打印内容：

// obj.a.call({})
{}
// obj.b.call({})
{}
// obj.c.call({})
Window {window: Window, self: Window, document: document, name: '', location: Location, …}

我们现在发现this的指向也如同我们熟悉的普通函数和箭头函数的区别一样，箭头函数无法直接更改this指向，随后我们尝试更改obj.c()的父级函数的this指向，当然现在的obj对象已经无法通过操作更改父级函数的this指向，我们需要重新创建函数func：

function func(){
  let obj = {
    a: function(){
      console.log(this)
    },
    b(){
      console.log(this)
    },
    c:()=>{
      console.log(this)  
    }
 }
 obj.a()
 obj.b()
 obj.c()
 obj.a.call({})
 obj.b.call({})
 obj.c.call({})
}

我们直接执行func()我们会发现以下结果：

{a: ƒ, b: ƒ, c: ƒ} // obj.a()
{a: ƒ, b: ƒ, c: ƒ} // obj.b()
Window {window: Window, self: Window, document: document, name: '', location: Location, …} // obj.c()
{} // obj.a.call({})
{} // obj.b.call({})
Window {window: Window, self: Window, document: document, name: '', location: Location, …} //obj.c.call({})

我们更改父级函数的this指向后，得到：

{a: ƒ, b: ƒ, c: ƒ} // obj.a()
{a: ƒ, b: ƒ, c: ƒ} // obj.a()
{} // obj.a()
{} // obj.b.call({})
{} // obj.b.call({})
{} // obj.b.call({})

我们发现更改对象内的箭头函数的父级对象的this指向时，箭头函数的this只想也会改变，而函数b与函数a的this指向可以直接修改。
